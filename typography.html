Visual Identity

Typography

The Recess type system uses Inter as its sole typeface, with a fluid scale that adapts from mobile to desktop. Every size is rendered live below.

Font Family

Inter is loaded from Google Fonts with weights 400 through 900. Add this import to any page that uses the design system.

@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800;900&display=swap');
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

Usage Rules

Typography Do's and Don'ts

  • Hero headlines always use weight 900 (Black). No exceptions. This is the signature Recess look.
  • Body copy always uses weight 400 (Regular). Never bold entire paragraphs.
  • Use Semibold (600) for inline emphasis within body text instead of bold.
  • Use italics sparingly and only for proper names, titles, or foreign phrases. Never for emphasis.
  • Section labels must be uppercase with wide tracking (0.12em) and aqua color.
  • Never go below 9px for any text. Accessibility requires minimum readable sizes.
  • All headline tracking is negative (tight). All label tracking is positive (wide). Never reverse these.
