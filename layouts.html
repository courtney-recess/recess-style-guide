Layout Patterns

Spacing, grid structures, content widths, and responsive breakpoints used across all Recess pages.

Section Structure

Every content section uses consistent vertical and horizontal padding with a centered max-width container.

Token Value Usage

--max-width-content 1160px Maximum width of content container
--spacing-section-y-mobile 60px Reduced vertical padding on mobile
--spacing-section-x-mobile 24px Reduced horizontal padding on mobile
CSS
.section {
  padding: var(--spacing-section-y) var(--spacing-section-x);
  max-width: var(--max-width-content);
  margin: 0 auto;
}

CSS
@media (max-width: 900px) {
  :root {
    --spacing-section-y: var(--spacing-section-y-mobile);   /* 60px */
    --spacing-section-x: var(--spacing-section-x-mobile);   /* 24px */
    --spacing-card-padding: var(--spacing-card-padding-sm);  /* 28px */
  }

  .card-grid {
    grid-template-columns: 1fr;
  }
}
