Usage Guidelines

When to Use Each Color

  • Aqua — Primary CTAs, links, interactive highlights, section labels, brand moments. This is the signature Recess color.
  • Sea Green — Hover states for aqua elements. Never used as a standalone brand color.
  • Forest — Success states, secondary accent. Use when aqua would cause visual overload.
  • Burnt Orange / Orange — Secondary CTAs, pricing highlights, urgency indicators. Use sparingly to avoid competing with aqua.
  • Navy palette — Page backgrounds on dark-theme sections. Navy is the default dark body background, Navy Light for elevated cards.
  • Greys — Grey 50 for light section backgrounds, Grey 200 for borders, Grey 500 for secondary text, Grey 800 for dark text on light backgrounds.
