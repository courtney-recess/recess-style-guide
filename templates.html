Most Used 
Build a Recess landing page with these exact specifications:

THEME: Dark (navy background #0a0f1a)
PARTNER: [Partner name or "Recess internal"]
GOAL: [Brand awareness / Lead generation / Case study / Event promotion]

- Section labels: 11px uppercase, letter-spacing 2px, aqua color
- Section titles: clamp(28px,3.5vw,44px) weight 800
- Max content width: 1160px centered
- Section padding: 100px vertical, 48px horizontal
- Card radius: 16px, button radius: 10px
- Card hover: border glow rgba(20,185,214,0.15) + translateY(-3px)
- Button hover: color shift + translateY(-1px) + shadow 0 0 32px rgba(20,185,214,0.40)
- All transitions: 0.2–0.4s ease
- Photography: always overlay (aqua 20% + grey 50%), never bare images
- Responsive: stack at 900px, adjust at 600px

REQUIRED SECTIONS:
1. Nav (fixed, glassmorphism, logo left, links + CTA right)
2. Hero (headline, subheadline, CTA button, 88vh)
3. At least 2 content sections (features, stats, cards, testimonials)
4. CTA section near bottom
5. Footer (logo, copyright, links)

OUTPUT: Single self-contained HTML file, all CSS inline except Google Fonts import.

Presentation Deck (Dark)

Slides 
Create a Recess presentation deck with these exact brand specifications:

FORMAT: Google Slides / PowerPoint
THEME: Dark
PURPOSE: [Partner pitch / Internal review / Case study / Event recap]
SLIDE COUNT: [Approximate number, e.g. 10-15]

BRAND RULES (non-negotiable):

COLORS:
- Slide background: #0a0f1a (navy)
- Primary accent: #14B9D6 (aqua) — used for highlights, key stats, icons, divider lines
- Secondary accent: #108DA3 (sea green) — used for subtle accents, secondary elements
- Text primary: rgba(255,255,255,0.85) — main body text
- Text secondary: rgba(255,255,255,0.60) — supporting text, captions
- Text muted: rgba(255,255,255,0.25) — footnotes, slide numbers
- Card/box backgrounds: rgba(255,255,255,0.03) with rgba(255,255,255,0.06) border
- Stat highlight numbers: #14B9D6 (aqua)
- Success/positive: #1FBBA6 (forest green)
- Warning/attention: #ff8900 (orange)
- NEVER use any color not listed above

TYPOGRAPHY:
- Font: Inter only — NEVER use any other font
- Slide title: 36-44px, weight 800, letter-spacing -0.5px
- Subtitle / section label: 11-12px, uppercase, letter-spacing 2px, aqua color
- Body text: 18-20px, weight 400, line-height 1.6
- Stat numbers: 48-64px, weight 900, aqua color
- Stat labels: 14px, weight 500, rgba(255,255,255,0.60)
- Captions/footnotes: 12px, weight 400, rgba(255,255,255,0.25)
- Bullet points: 16-18px, weight 400, use aqua bullets or dashes

LAYOUT:
- Widescreen 16:9 aspect ratio
- Content margins: 80px from edges minimum
- Consistent padding across all slides
- Left-aligned text (never centered body text, titles may center on title slides)
- Max 6 bullet points per slide
- Max 30 words per bullet
- One idea per slide — don't overcrowd

LOGO:
- Title slide: Recess logo centered or top-left, white version on dark bg
- Internal slides: small white logo in bottom-left or bottom-right corner
- NEVER recreate logo as text — always reference the actual logo file
- NEVER alter, rotate, warp, or recolor the logo

SLIDE TYPES TO INCLUDE:
1. Title slide (logo, deck title, date, subtitle)
2. Agenda / overview slide
3. Section divider slides (aqua accent line, section name in uppercase)
4. Content slides (text + supporting visual)
5. Stat/metric slides (big number + context)
6. Comparison slides (side-by-side or before/after)
7. Quote/testimonial slides
8. Closing slide (CTA or next steps + logo)

PHOTOGRAPHY & IMAGERY:
- Any photos should have a tinted overlay (aqua 20% + dark 50%)
- Never use bare/unfiltered photos
- Prefer simple icons over complex illustrations
- Icons should be white or aqua, thin-line style

DESIGN DETAILS:
- Use subtle aqua divider lines (1-2px) to separate content sections
- Rounded corners (12-16px) on any boxes, cards, or image frames
- No drop shadows — use border/outline treatment instead
- Consistent spacing between elements (24px minimum gaps)
- Slide numbers in bottom-right, 12px, muted text color

CONTENT TO USE:
- Deck title: [Your title]
- Audience: [Who is this for]
- Key message: [The one thing they should remember]
- Slides needed: [Describe each slide's content]
Best for: Partner pitches, investor decks, team presentations. Navy background with aqua accents — same visual DNA as the dark landing page.

Email / One-Pager

Multi-Use 
Write Recess-branded content with these exact brand voice and formatting rules:

FORMAT: [Marketing email / Partner one-pager / Sales leave-behind / Newsletter]
AUDIENCE: [Who is reading this]
GOAL: [What action should they take after reading]

BRAND VOICE (non-negotiable):
- Personality: playful, confident, inclusive, wellness-forward
- Tone: casual but informed, optimistic without hype, direct not corporate
- Active voice always — never passive
- Short sentences preferred — keep paragraphs to 2-3 sentences max
- Plain language — no jargon, no buzzwords, no "synergy" or "leverage"
- Contractions are fine (we're, it's, don't)
- Never use exclamation marks more than once per piece
- Headlines should be punchy and benefit-driven, not clever for clever's sake

FORMATTING:
- Headline: bold, concise, benefit-first (max 8 words)
- Subheadline: one sentence expanding on the headline
- Body: scannable — use short paragraphs, bullet points, bold key phrases
- CTA: clear single action, direct language ("See the results" not "Click here to learn more")
- Stats: lead with the number, make it big and bold (e.g., "42% increase" not "We saw a forty-two percent increase")

COLOR REFERENCES (for any HTML email or designed one-pager):
- Primary accent: #14B9D6 (aqua) — CTA buttons, links, highlights
- Headlines: #000000 on light / rgba(255,255,255,0.85) on dark
- Body text: #767676 on light / rgba(255,255,255,0.60) on dark
- Background: #ffffff (light) or #0a0f1a (dark)
- Button: #14B9D6 background, white text, rounded corners

CONTENT TO USE:
- Topic: [What is this about]
- Key points: [Main messages to convey]
- CTA: [What should the reader do next]
- Stats/proof points: [Any numbers or results to highlight]
Best for: Marketing emails, partner one-pagers, sales leave-behinds, newsletters. Ensures the Recess voice and formatting stays consistent across written content.

How to Use Templates

Follow these steps to create a new landing page from a template.

  1. Choose a template (Dark, Light, or Partner) based on the project requirements
  2. Open the template file and save a copy with your project name
  3. Search for CUSTOMIZE comments to find all placeholder content
  4. Replace placeholder text with your actual content (headlines, descriptions, stats)
  5. For partner templates, replace the accent color as noted in the PARTNER COLOR comments
  6. Add real images and update the navigation links
  7. Test at desktop, tablet (900px), and mobile (600px) breakpoints
