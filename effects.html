Glassmorphism

Frosted glass effect using backdrop-filter blur with semi-transparent backgrounds. Used for navigation bars and overlays.

Glassmorphism Panel

backdrop-filter: blur(12px) over colorful background

CSS
background: var(--glass-bg-dark);    /* rgba(10, 15, 26, 0.75) */
backdrop-filter: var(--glass-blur);   /* blur(12px) */
-webkit-backdrop-filter: var(--glass-blur);
border: 1px solid rgba(255, 255, 255, 0.15);

Hover Lift

Cards and buttons lift upward on hover with translateY(-3px). Paired with a shadow increase for depth.

Hover the card below

Hover Me

This card lifts translateY(-3px) on hover with a subtle shadow.

CSS
transition: transform var(--transition-base),
            box-shadow var(--transition-base);

&:hover {
  transform: var(--hover-lift);       /* translateY(-3px) */
  box-shadow: var(--shadow-card);     /* 0 4px 24px rgba(0,0,0,0.08) */
}

Image Zoom

Images scale to 1.04 on hover within an overflow:hidden container for a subtle zoom-in effect.

Hover the image below
Hover to zoom (scale 1.04)
CSS
.image-wrapper {
  overflow: hidden;
  border-radius: 12px;
}

.image-wrapper img {
  transition: transform var(--transition-base);
}

.image-wrapper:hover img {
  transform: var(--hover-scale);  /* scale(1.04) */
}

Button Transitions

Primary buttons shift from aqua to sea-green on hover, with the glow shadow intensifying. Ghost buttons gain a subtle fill.

Hover the buttons below
CSS Tokens
/* Primary button */
background: var(--color-primary);                /* #14B9D6 */
box-shadow: var(--shadow-cta-glow);              /* 0 0 24px rgba(20,185,214,0.25) */

/* On hover */
background: var(--color-primary-hover);          /* #108DA3 */
box-shadow: var(--shadow-cta-glow-hover);        /* 0 0 32px rgba(20,185,214,0.40) */
transform: var(--hover-lift);                    /* translateY(-3px) */
transition: var(--transition-button);            /* 200ms ease */

Marquee Scroll

Auto-scrolling horizontal text that loops infinitely over 30 seconds. Used for brand partner logos or feature lists.

Recess Relaxation Wellness Balance Calm Mindfulness Serenity Focus Recess Relaxation Wellness Balance Calm Mindfulness Serenity Focus
CSS
@keyframes marquee-scroll {
  0%   { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}

.marquee-track {
  display: flex;
  gap: 48px;
  animation: marquee-scroll var(--animation-marquee) linear infinite;
  /* --animation-marquee: 30s */
}
