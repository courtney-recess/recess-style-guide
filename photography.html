Photography Guidelines

Image selection, overlay treatments, and art direction rules for maintaining brand consistency across all visual content.

Image Selection Guidelines

Follow these rules when selecting photography for Recess marketing materials and landing pages.

✓

Do

  • ✓ Real people in real places — authentic, unposed moments
  • ✓ Highest professional standards — sharp focus, proper exposure
  • ✓ Overall lightness in tone — bright, airy, optimistic feel
  • ✓ Consistent with brand personality — calm, balanced, joyful
  • ✓ Lifestyle, events, and activations imagery
  • ✓ Happy, fun, excited, and active people
✕

Don't

  • ✕ Cluttered compositions — too many elements competing for attention
  • ✕ Irrelevant to marketing message — off-brand or confusing imagery
  • ✕ Low quality or stock-looking — generic, staged, or low-resolution
  • ✕ Heavy or dark tonality without overlay treatment

Overlay Technique

The brand overlay technique uses two layers on top of photography to create a consistent, on-brand look.

CSS Overlay Implementation
/* Image with brand overlay treatment */
.brand-image {
  position: relative;
  overflow: hidden;
}

/* Layer 2: Aqua tint */
.brand-image::before {
  content: '';
  position: absolute;
  inset: 0;
  background: var(--overlay-aqua);   /* rgba(20, 185, 214, 0.20) */
  z-index: 1;
}

/* Layer 3: Grey depth */
.brand-image::after {
  content: '';
  position: absolute;
  inset: 0;
  background: var(--overlay-grey);   /* rgba(0, 0, 0, 0.50) */
  z-index: 2;
}

/* Content on top of overlays */
.brand-image .content {
  position: relative;
  z-index: 3;
}

Logo on Images

Always use the white version of the Recess logo when placing it over images. Apply the overlay treatment first to ensure legibility.

White logo + overlay = legible
Without overlay = poor contrast
Correct: With overlay treatment
Incorrect: Without overlay treatment
Guidelines
/* Logo placement rules:
 *
 * 1. Always use WHITE version of logo on images
 * 2. Apply overlay treatment FIRST for legibility
 * 3. Logo should have minimum 24px padding from edges
 * 4. Never place logo on busy/cluttered areas
 * 5. Test contrast: logo must be clearly readable
 */
